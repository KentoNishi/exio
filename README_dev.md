## Clone the repository

```bash
git clone git@github.com:KentoNishi/exio.git
```

## Install dependencies

```bash
cd exio
yarn
```

## Link the package

```bash
cd src/lib
yarn link
cd ../../
yarn link exio
```

## Run the dev build

```bash
yarn dev
```

## Build the package

```bash
cd src/lib
yarn build
```
